## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need Postgres:

 On Linux:
 
```sh
 sudo apt-get install postgresql postgresql-contrib libpq-dev
```

On Windows

Go to [postgres website](https://www.postgresql.org/) download the installer and follow the installation process

### Setup

Clone this repository to your desired folder:

```sh
  git clone git@github.com/Stephen-Adom/vet-clinic-database
  cd vet-clinic-database
```
